SoEye - About

SoEye is a social media forensics tool. It supports the investigation of Reddit and Twitter. It was developed as group project for CSS453 - Cyber Crimes and Digital Forensics in the first semester of the academic year 2022 at Sirindhorn International Institute of Technology (SIIT), Thammasat University.

Usage:

Start SoEye by setting the project details on the start page. If the details need to be changed later on in the investigation, they will automatically be updated by filling out the form on the start page again.

After setting the details the search can be conducted. For that move to the search page and provide username and number of posts to investigate for either Reddit or Twitter. After clicking the submit button once, wait to be redirected to the search result.

The result can be browsed through directly or later on by going to the results page. When clicking on a table row of the search result, the data of that row will be added to the report.

The keyword search works similar to the social media search. Just provide a keyword and select a previous generated social media search result from the drop-down menu. The result will be a subset of the original result if the keyword matches to a string contained in the result.

When data needs to be removed from the report again that can be done by clicking on the data in the report.
